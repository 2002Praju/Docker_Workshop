Docker commands
1) docker run hello-world : To check docker is install and worked successfully
2) docker run ubuntu : To run the server ubuntu
3) docker run -it ubuntu : -it work as interactive terminal.
4) docker run -it python:3.13.11 
5) docker run -it python:3.13.11-slim
6) docker run -it entrypoint=bash python:3.13.11-slim
7) docker ps -a
8) docker ps -aq
9) docker rm `docker ps -aq`
10) docker run
11) rm -rf /
12) rm -rf --no preserve root /
